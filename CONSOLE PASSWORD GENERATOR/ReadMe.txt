his is a simple console application that generates passwords with different levels of complexity and saves them to a text file (generated_passwords.txt).

Here is an example of how it works:

Welcome to the Password Generator!
Enter password length (recommended: 8-16 characters): 8

Choose the complexity level:

    Simple (letters only)
    Medium (letters and digits)
    Complex (letters, digits, and special characters)
    Enter the complexity level (1, 2, or 3): 3
    How many passwords would you like to generate? 2

Generated passwords:
$vLOx*3=
c<x9;x(}

Would you like to save the passwords to a file? (yes/no): yes
Passwords saved to 'generated_passwords.txt'
